# Disaster Response & Path Finding System
A C++ console-based application designed to simulate disaster management operations.
It allows reporting victims, dispatching rescue teams, managing road blockages, and visualizing city map status using data structures and algorithms.

## Features
- Report New Victim: Log emergency calls into the system.
- Dispatch Rescue Team: Assign available teams to victims.
- Add New Rescue Team: Register new rescue units.
- Team Returned to Base: Update team availability after missions.
- Manage Road Blockages: Track and update blocked routes.
- View Pending Victims (Priority Queue): Handle victims based on urgency.
- View Rescue Teams (BST): Organize and display teams using Binary Search Tree.
- View City Map Status (Graph): Visualize city routes and blockages.
- View System Logs (Linked List): Maintain chronological logs of system activities.

## Tech Stack
- Language: C++
- Concepts Used:
   - Priority Queue (Victim management)
   - Binary Search Tree (Rescue teams)
   - Graphs (City map & pathfinding)
   - Linked List (System logs)
